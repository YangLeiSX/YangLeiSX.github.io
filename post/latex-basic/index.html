LaTeX基础部分简单记录

Lei Yang
Oct 10, 2019 1 min read 实用工具

LaTeX基础部分的使用方法

文章框架

% 指定文章类型
\documentclass[a4paper]{article}
% ctex为中文环境
\documentclass[a4paper]{ctexart}
% 如不支持则使用
% \usepackage{ctex}

% 指明作者
\author{}
% 文章标题
\title{}
% 指定时间
\date{}  
% 可用参数 \today 括号内什么都不写则标题里不会写时间

% 文章开始
\begin{document}
% 文章结束
\end{document}

% 写标题
\maketitle
% 目录
\tableofcontents
% 章节1
\section{}
% 章节1.1
\subsection{}
% 章节1.1.1
\subsubsection{}
% 段落开头加粗名词介绍
\paragraph{名词}
% 此段落加粗名词介绍
\subparagraph{名词}
% 换行
\\
% 翻页
\newpage
% 加粗黑体
\textbf{}
% 垂直间距
\vskip1.0cm

添加目录树

超链接包\usepackage{hyperref}

可以在pdf中生成目录树和超链接 点击目录和参考文献可以自动跳转

默认情况下超链接通过红色边框标明 Colorlinks命令可以使超链接通过颜色标明 后接linkcolor=red citecolor=green可以指明不同种类链接的颜色

如果不想有颜色可以指明颜色为black，超链接功能仍适用 该方法可以不显示红色边框

列举条目

列举包\usepackage{enumerate}

条目环境\begin{itemize} \end{itemize} ​每一条目用\item （必须加空格）

列举环境\begin{enumerate}[参数] \end{enumerate} ​也需要用\item (加空格) ​参数标明序号格式 例 [part 1][i][<1>]

页面边框

边框包\usepackage{geometry}

​指定边框\geometry{left = cm, right = cm, top = cm, bottom = cm}

局部字体大小 \Huge \huge \LARGE \large \normalsize \small \footnotsize \scriptsize \tiny

单页竖排

使用landscape包 \usepackage{lscape}

\begin{landscape}\end{landscape}

居中环境

\begin{center}\end{center}

或者在环境内加入 \centering

页码设置

页码样式\pagestyle{type}

本页面样式\thispagestyle{type} 无页码empty 无页眉页脚为页码plain 无页脚页眉有章节和页码headings 无页脚页眉只有页码myheadings

手动设置页码\setcounter{page}{number} 页码风格\pagenumberingstyle{style} 数字arabic 小写罗马字roman 大写罗马字Roman 小写字符alph 大写字符Alph
