花书第三章:概率与信息论

Posted on 2020-02-20 Edited on 2020-05-19 In 读书笔记
Symbols count in article: 5.7k Reading time ≈ 5 mins.

概率论能够提出不确定的声明以及在不确定性存在的情况下进行推理,信息论可以量化概率分布中的不确定性总量。

为什么使用概率

机器学习经常需要处理不确定量和随机量。

不确定性来源：被建模系统内在的随机性、不完全观测、不完全建模。

使用简单而不确定的规则要比复杂而确定的规则更加实用。

与事件发生频率相联系的成为频率派概率，与事件发生的确定性水平有关的（概率表示信任度）为贝叶斯概率。

随机变量

可以随机取值的变量。

离散的或者连续的。

概率分布

描述随机变量在可能取到的状态的可能性大小。

离散随机变量和概率质量函数

概率质量函数PMF或者称为概率分布函数，将随机变量的状态映射到概率。

作用于多个变量的是联合概率分布函数。

注意归一化条件$\sum\limits_{x \in x} P(x) = 1$。

连续随机变量和概率密度函数

概率密度函数PDF描述连续型随机变量。

边缘概率

从联合概率计算某一个变量子集的概率分布

使用求和$P(x) = \sum\limits_{y}P(x,y)$或者积分$p(x) = \int p(x,y)dy$。

条件概率

$P(y={y}|x=x) = \frac{P(x=x,y=y)}{P(x=x)}$

条件概率的链式法则

或者称为乘法法则

$P(abc) = P(a|bc)P(b|c)P(c)$

独立性和条件独立性

有$P(xy) = P(x)P(y)$称为概率独立，即$x \bot y$。

有$P(xy|z) = P(x|x)P(y|z)$称为条件独立，即$x\bot y\ |\ z$。

期望、方差和协方差

期望是$\mathbb{E}[f(x)] = \int p(x)f(x)dx$或者$\mathbb{E}[f(x) = \sum \limits_{x}P(x)f(x)]$。

方差为$Var(f(x)) = \mathbb{E}[(f(x) - \mathbb{E}[f(x)])^2]$，根式为标准差。

协方差衡量多个变量相关性$Cov(f(x), g(y)) = \mathbb{E}[(f(x) - \mathbb{E}[f(x)])\times(g(y) - \mathbb{E}[g(y)])]$。

随机向量的协方差矩阵有$Cov(\mathbf{x})_{i,j} = Cov(x_i, x_j)$。对角线为方差。

常用的概率分布

Bernoulli分布

二值分布$P(x=1)=\phi$、$P(x=0)=1-\phi$

Multinoulli分布

称为范畴分布categorical distribution，其实是多项式分布（N个元素取n次）的n=1的情况。

表示有限个不同状态的离散随机变量的分布

高斯分布

称为正态分布normal distribution。

$\mathcal{N}(x;\mu,\sigma^2) = \sqrt{\frac{1}{2\pi\sigma^2}}exp(-\frac{1}{2\sigma^2}(x-\mu)^2)$

通常使用$\beta = \frac{1}{\sigma^2}$作为高斯分布的精度precision

重要性：独立随机变量的和近似服从正态分布；正态分布是引入先验知识量最少的分布。

可以推广到多维正态分布$\mathcal{N}(x;\mu,\Sigma)=\sqrt{\frac{1}{(2\pi)^ndet(\Sigma)}}exp(-\frac{1}{2}(x-\mu)^T\Sigma^{-1}(x-\mu))$。

其中的$\mu$表示均值，$\Sigma$表示协方差矩阵。

指数分布和Laplace分布

指数分布，其中的指示函数$ 1_{x\geq0} $将自变量范围限制在不为负数。

Laplace分布$Laplace(x;\mu, \gamma) = \frac{1}{2\gamma}exp(-\frac{|x-\mu|}{\gamma})$。

Dirac分布和经验分布

当概率质量集中在一点时得到Dirac分布$p(x) = \delta(x-\mu)$。可以表示一次观察的结果。

经验分布$\hat{p}(\mathbf{x}) = \frac{1}{m}\sum \limits_{i=1}^{m} \delta(\mathbf{x} - \mathbf{x}^{(i)})$。可以表示对样本的m次观察，或者规模为m的训练集。

训练模型时，看作用训练集的经验分布指明采样来源的分布。

经验分布可以看作训练数据似然最大的概率密度函数。

分布的混合

混合分布mixture dixtribution由多个分布成分component组成，首先每个成分来自Multinoulli分布。

$P(x) = \sum \limits_{i} P(c=i)P(x|c=i)$其中$P(c)$是各个组成成分的一个Multinoulli分布。

引入了潜变量的概念，即不能直接观测到的变量，但是观测变量受到潜变量的影响，两者的分布共同影响最终分布。

常见高斯混合模型，每一个成分都是均值和协方差矩阵各不相同的高斯分布。其中给定了$P(c=i)$是每一个高斯分布的成分的先验概率，而$P(c=i｜x)$称为后验概率。

高斯混合模型是万能近似器，任何平滑的概率密度都可以用足够多成分的高斯混合模型以任意精度来逼近。

常见函数的有用性质

Logistic Sigmoid函数：$\sigma(x) = \frac{1}{1+exp(-x)}$。

Softplus函数：$\zeta(x) = log(1+exp(x))$。

其中Softplus函数可以看作是正部函数的平滑版本，正部函数$x^+ = max{0, x}$就是常用的ReLU激活函数。

贝叶斯规则

贝叶斯规则：$P(x|y) = \frac{P(x)P(y|x)}{P(y)}$。

其中可以计算$P(y) = \sum\limits_{x}P(y|x)P(x)$。

连续型变量的技术细节

连续型随机变量和概率密度函数的简单定义可能会引起悖论，因此使用测度论的一些知识加以处理。

提出“零测度”的集合、“几乎处处”成立的概念。

同时对于具有函数关系的随机变量的分布的处理不能简单的使用微分的概念处理。

信息论

定义自信息量$I(x) = -logP(x)$。

定义信息熵（香农熵）。

定义KL测度，表示两个概率分布的差异。注意由于X服从的分布不同，KL测度是不对称的。

定义交叉熵（cross entropy）。

结构化概率模型

涉及在很多个相互影响的随机变量上的概率分布时，可以按照依赖和独立关系将其分解成多个概率分布的乘积。

通常使用图模型表示多个变量之间的关系，称为结构化概率模型或者图模型。

有向图模型中，使用有向边表示条件概率分布。$p(x) = \prod\limits_{i}p(x_i|Pa\mathcal{G}(x_i))$其中$Pa\mathcal{G}(x_i)$表示$x_i$的父节点。

无向图模型中，将图分为团（团内部任意两节点之间存在无向边）并带有因子，整个概率分布表示为各个团对应的函数按因子加权和并且归一化$p(x) = \frac{1}{Z}\prod\limits_{i}\phi^{(i)}(\mathcal{C}^{(i)})$。
