我的Windows电脑装了哪些软件

Posted on 2020-09-12 In 实用工具
Symbols count in article: 4.2k Reading time ≈ 3 mins.

最近抽空整理了一下电脑里面装了一些什么软件，毕竟没有备份，如果机器出了什么问题，重新装机时候也有个参考，另一个原因也算是推荐一些好用的小工具吧。
从Mac选手换到Windows刚开始有点不适应，大半年过去了现在觉得习惯了还挺方便的。

办公软件
Adobe Acrobat DC PDF 阅读和编辑工具
Microsoft Office（Word、Powerpoint、Excel） 众所周知的办公套件
Microsoft OneDrive 微软云盘
Typora 众所周知的Markdown 编辑器
编程相关
MinGW C/C++ 编译环境
Anaconda Python 包管理环境
Java（JDK） Java 开发环境
Clion 用于C/C++ 的IDE
IntelliJ IDEA 用于Java 的IDE
PyCharm 用于Python 的IDE
Qt（Qt Creator） 开发Qt 程序的IDE 和环境
MATLAB 众所周知的数学处理程序
Microsoft Visual Studio 宇宙第一IDE
Microsoft Visual Studio Code 比较常用的编辑器
VMware Workstation 虚拟机管理软件
Wireshark 抓包和网络分析工具
texlive LaTeX 编译环境
Windows Terminal Preview Windows 原生终端
Windows Subsystem for Linux（Ubuntu） Windows 子系统
WinSCP 可视化的SCP 文件传输工具
Git 众所周知的代码管理工具
gVim 众所周知的编辑器
Scoop Windows 环境的包管理工具
其他工具
BitTorrent BT 文件下载工具
Motrix 一个开源的下载器
百度网盘 众所周知的网盘
Clash 代理服务器管理工具
ShareX 截图工具
Wox 文件搜索和启动器(结合Everything 使用)
Everything 文件搜索工具
PowerToys 微软官方提供的一套工具
Stardock Fences 桌面图标管理工具
GeekUninstaller 非常好用的卸载工具
RealTemp 实时温度检测工具
Shadowsocks 众所周知的代理服务器管理
v2rayN 代理服务器管理工具
SysinternalsSuite 微软官方的系统管理工具
Microsoft To Do 日程和待办清单管理
QuickLook 空格键预览工具
FileZilla FTP 文件传输工具
火绒安全 杀毒和安全管理工具
欧陆词典 非常好用的词典
ZOOM 视频会议工具
腾讯会议 视频会议工具
Google Chrome 众所周知的浏览器
PotPlayer 一个比较好用的播放器
Logitech G HUB 罗技设备管理工具
Teamviewer 远程登录和管理工具
VNCServer 远程登录和管理工具
维护不易，请杨磊喝杯奶茶吧。
